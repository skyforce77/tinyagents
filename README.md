# tinyagents

A tiny, modular agent-actor platform for Go — a minimalist actor library meets LLM orchestration.

Compose heterogeneous LLM agents (Ollama, Anthropic, OpenAI, Mistral, OpenRouter)
into teams with typed messaging, backpressure, supervision, routing, CRDTs, and
gossip-based clustering. Each agent can use its own provider and model.

## Status

Early — v0.1.0 under active development. APIs will change.
